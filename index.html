nimble.css

A minimal classless CSS library. Drop it in and your semantic HTML looks good — no classes required. ~3.3 KB brotli (core).

Demos

  • HTML5 Test Page — every standard HTML element
  • Extended Demo — layouts, utilities, button variants, dark mode toggle

Features

  • Classless by default — styles every standard HTML element with zero markup changes
  • Automatic dark mode — via prefers-color-scheme and light-dark()
  • OKLCH color system — perceptually uniform, fully customizable
  • Cascade layers — plays nicely alongside your own styles
  • Tiny — core is 15.4 KB minified, ~3.3 KB brotli; full bundle is 18.9 KB minified, ~4 KB brotli

Quick Start

Add one line to your <head>:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@leftium/nimble.css">

Or install via npm:

npm install @leftium/nimble.css
/* CSS */
@import '@leftium/nimble.css';

// JS / framework entry point
import '@leftium/nimble.css';

That's it. Write semantic HTML and it just works.

Bundles

nimble.min.css (18.9 KB) includes everything.

To trim size, use nimble-core.min.css (15.4 KB) + only the add-ons you need:

Add-on Size
nimble-progress.min.css 1.6 KB
nimble-meter.min.css 1.0 KB
nimble-select.min.css 1.0 KB

Customization

Override CSS custom properties at runtime — no build step needed:

:root {
  --nc-primary: oklch(0.50 0.15 150);
  --nc-radius: 0.5rem;
  --nc-spacing: 1.25rem;
}

Or use SCSS for build-time customization:

@use 'nimble' with (
  $primary-hue: 150,
  $primary-chroma: 0.15,
  $content-width: 960px,
);

See the README for the full list of config variables.
